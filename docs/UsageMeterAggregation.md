# UsageMeterAggregation

Aggregation to be applied on usage meter result * COUNT - Counts number of events matching the usage meter * SUM - Sums up results of computation of all events matching usage meter 

## Enum

* `COUNT` (value: `'COUNT'`)

* `SUM` (value: `'SUM'`)
